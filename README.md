# Mini Proyecto: Coin Collector

## Assembling and Linking Instructions
Utilities for building the project are included under the `**/scripts**` folder.
These utilities must be executed from the project's root folder (i.e., from `**Mini_proyecto/**`).

### For Windows:
```
powershell -ExecutionPolicy Bypass -File scripts\build.ps1
```

### For MacOS and Linux:
Make sure the script has execution permissions: `chmod +x scripts/build.sh`
```
./scripts/build.sh
```
